# Employee Salary Prediction

### Adding New Skills
To add new skills to the prediction engine:

1. Open `src/utils/predictionEngine.ts`
2. Add the skill to the `SKILL_BONUSES` object with appropriate bonus value
3. Update the `AVAILABLE_SKILLS` array in `src/components/PredictionForm.tsx`

### Modifying Multipliers
Adjust salary multipliers in `src/utils/predictionEngine.ts`:
- `EDUCATION_MULTIPLIERS`
- `LOCATION_MULTIPLIERS`
- `DEPARTMENT_MULTIPLIERS`

### Styling Changes
The application uses Tailwind CSS. Modify classes in component files or extend the configuration in `tailwind.config.js`.

- **Base Accuracy**: 85% confidence level
- **Enhanced by**: Number of skills (+2% per skill)
- **Experience Bonus**: +5% for experienced professionals
- **Maximum Confidence**: 98%
